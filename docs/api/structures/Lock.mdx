# Lock

The `Lock.create()` function returns a promise and a function to resolve and
another one to reject it. It is the same as `EjectedPromise<void>`.

<Tabs>
  <TabItem value="tuple" label="Tuple" default>

```ts
import { Lock, fork } from "@alfonz/async";

const [promise, resolve] = Lock.create<void>();

fork(async () => {
	await promise;
	console.log("world");
});

console.log("hello");
resolve();
```

  </TabItem>
  <TabItem value="object" label="Object">

```ts
import { Lock, fork } from "@alfonz/async";

const ejected = Lock.create<void>();

fork(async () => {
	await ejected.promise;
	console.log("world");
});

console.log("hello");
ejected.resolve();
```
